點解要用粵拼打字？

每當你用粵拼打字，你嘅腦海入面就要出現一次粵語嘅標準發音然後輸出，通過呢個過程你就會不斷鍛鍊自己嘅粵語標準音。無論係母語者定係學習者，都可以通過拼音打字嚟練習自己嘅發音。

普通話人用普通話拼音打字，粵語人就用粵語拼音打字。

  2. 我本身已經用緊 Rime 啦噃，噉要點樣加粵語拼音落 Rime 度？

    1. 用 + Space 切換去小狼毫輸入法。
    2. 喺個「中」字上面右擊，揀 輸入法設定。
    3. 撳 獲取更多輸入方案…。
    4. 然後喺彈出嚟個命令行界面，輸入 cantonese，然後撳 Enter。跟住分別輸入 emoji 同埋 sgalal/rime-opencc-32bit-latest 嚟安裝 Emoji 套件同埋最新版嘅繁簡轉換套件。
    5. 閂咗「Rime package installer」個視窗佢，去返「輸入法設定」個版面，勾選「粵語拼音」，然後撳 中。
    6. 跟住揀返自己中意嘅配色，再撳 中 嚟完成設定
    7. 去返選單度撳 重新部署，等個程式處理詞庫資料。
    8. 跟住撳 Ctrl + `（注意係數字 1 左邊嗰粒掣，唔係單引號！），就可以睇到「粵語拼音」嘞。
